### Aufgabe 1

Probiere die folgenden Anweisungen aus, um Textvariablen oder **Strings** zu erzeugen:

```python
vorname = "Ada"
nachname = "Lovelace"
name = vorname + " " + nachname
print(name)
```

### Aufgabe 2

```python

```
